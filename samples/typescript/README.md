## Prerequisite

Make sure you have read the [prerequisite-guide](../README.md#prerequisite) before continuing.

## Install

Go to this folder in your terminal and run:

`$ npm install`

or with yarn:

`$ yarn`

## Build
`$ npm run tsc`

or with yarn

`$ yarn tsc`

## Run

`$ COGNITE_CREDENTIALS="YOUR_API_KEY" node build/quickstart.js`
